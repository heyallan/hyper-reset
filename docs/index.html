CSS hyper reset

Purpose

This library was written for styling static pages, like stand-alone hand-made HTML pages, or pages genterated with markdown, where it's impossible to inject CSS classes.

This library makes it possible to style whole pages created with text-to-html generators.

This library IS NOT intended to replace the use of CSS classes, but to provide a presentable layer of elements, similar to a reset. Use where appropriate.

Installation

Download:

https://unpkg.com/hyper-reset/dist/hyper-reset.min.css

CDN Development (always latest):

<link rel="stylesheet" href="https://unpkg.com/hyper-reset/dist/hyper-reset.min.css"/>

CDN Production (specific release):

<link rel="stylesheet" href="https://unpkg.com/hyper-reset@1.1.1/dist/hyper-reset.min.css"/>

NPM package:

$ npm install --save-dev hyper-reset

CSS import:

@import url("https://unpkg.com/hyper-reset@1.1.1/dist/hyper-reset.min.css");
